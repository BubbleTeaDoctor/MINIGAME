# 无限制角斗场 Demo

这是一个为《无限制角斗场》制作的首个电子游戏 Demo。

## 当前实现
- 单机 1v1
- 六角格地图
- 开局五模块构筑：职业 / 武器 / 饰品 / 种族 / 守护神
- 抽牌、出牌、普通攻击、格挡、结束回合
- 黑洞中心牵引、尖刺地形、陷阱
- 数据驱动模块结构，方便继续扩展职业、武器、饰品与守护神

## 当前内置内容
- 职业：战士、法师
- 武器：巨剑、长弓
- 饰品：林肯法球、口袋陷阱
- 种族：兽人、精灵、矮人
- 守护神：野蛮人、莉莉丝、欺诈者

## 运行方式
直接用浏览器打开 `index.html` 即可。
